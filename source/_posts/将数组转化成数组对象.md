### 前言
在实际开发中，我们经常需要对数据进行相应的处理，以便使用，这篇文章主要讲述如何将数组转换为数组对象格式
>原始数据：  ["苹果", "西瓜", "香蕉", "梨子"]

>需要转换为的数据： [{…}, {…}, {…}, {…}] 
0: {id: 0, name: "苹果"}
1: {id: 1, name: "西瓜"}
2: {id: 2, name: "香蕉"}
3: {id: 3, name: "梨子"}

>id，name可根据具体需求修改
### 方法一
```
let arr = ["苹果", "西瓜", "香蕉", "梨子"];
let obj = {};
// 将数组转化为对象
for (let key in arr) {
  obj[key] = arr[key];
}
newObj = Object.keys(obj).map((val) => ({
  id: val,
  name: obj[val],
}));
console.log(newObj);
```
### 方法二
```
let arr = ["苹果", "西瓜", "香蕉", "梨子"];
let newArr = []
for(let i = 0, len = arr.length; i < len; i++) {
  let obj = {};
  obj.id = i;
  obj.name = arr[i];
  newArr.push(obj)
}
console.log(newArr)
```
